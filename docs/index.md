## Features

### 💰 Save Money
- **Exact Prompt Caching** — SHA256-based deduplication
- **Semantic Caching** — ML-powered similarity matching
- **14-Model Pricing** — Accurate cost estimates

### 🛡️ Control Usage
- **Rate Limiting** — Per-API-key request caps
- **Budget Management** — Per-user monthly budgets
- **Anomaly Detection** — Detect cost spikes

### 📊 Stay Informed

- **Email Alerts** — SMTP-based alerts
